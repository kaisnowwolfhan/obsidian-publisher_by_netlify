# Core
培養兼具「會計」與「資訊」雙領域專長（Double Skills）
可選擇職涯發展如下：  
1）企業或銀行中高階會計及資訊人員：從事財務、成本、管理會計、稅務、預算及資訊系統、電腦稽核等工作  
2）政府人員：從事公務會計、政府審計工作  
3）大學教授：從事會計或會計資訊相關教學與研究
# 必修
一年級：初級==會計學==(一)、初級會計學(二)、經濟學原理(一)、==微積分==、==統計學==、==計算機概論==
二年級：中級會計學(一)、中級會計學(二)、成本及管理會計學(一)、成本及管理會計學(二)、==商事法、租稅法規==、==資料庫管理、程式設計==
三年級：中級會計學(三)、高級會計學(一)、高級會計學(二)、審計學(一)、==企業資源規劃、財務報表分析、永續發展與綠色會計==
四年級：審計學(二)、電腦審計、會計資訊系統、人文與企業倫理專題
# 選修
財務會計：初級會計學、中級會計學、高級會計學
管理會計：成本與管理會計學
審計：審計學、電腦審計、會計資訊系統
資訊學科：人工智慧導論、系統分析與設計、資料庫管理、企業資源規劃、資訊安全與管理、大數據與資料分析
