# CORE
加強學生對於統計軟體與程式語言的嫻熟度。包括資料的收集、整理、歸納，分析和建立模型以發掘問題、了解因果關係和推論真相，再進一步解決問題和預測未來。統計學中統計量的機率分配的知識在數據科學中非常重要。
# 必修
## 一年級
線性代數(3/3)、==經濟學==(3/3)、微積分甲(3/3)、統計學(一)(3/0)、統計學(二)(0/3)
## 二年級
==程式設計與統計軟體==(3/0)、數理統計學(一)(0/3)、初級==會計學==(一)(3/0)、迴歸分析(一)(0/3)、機率論(3/0)
## 三年級
數理統計學(二)(3/0)、抽調&變分實設&時數&多變量(四選三)、八管選一管(3)、==企業倫理與社會責任==(0/1)
∴八管為==管理學/商業分析/資訊管理==、==行銷管理/財務管理/營運與供應鏈管理==、==風險管理/人資管理==等
∴必修四選三之另一門作為專業選修
