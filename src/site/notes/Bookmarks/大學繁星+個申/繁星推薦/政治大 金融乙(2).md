# CORE
只要是經濟體系內資金創造、分配、使用與交易的相關活動都是金融學科的研究範疇。金融專業工作經常需要==分析大量的數據與資料==，因此數理能力與資訊科技應用能力優秀的自然組同學也很適合就讀金融系。
# 必修
## 一年級
==統計學(一)/(二)、微積分==甲、經濟學、初級==會計學==(一)、初級會計學(二)
## 二年級
個體經濟學、總體經濟學、==財務管理==、==貨幣銀行學、投資學、商事法==
## 三年級
衍生性金融商品、國際金融、金融法規/貨幣理論與政策/金融機構管理、==企業倫理與社會責任==
## 四年級
==銀行經營管理==
# 選修
#### 基礎科學領域
貨幣理論與政策
#### 金融市場領域
銀行經營管理、金融機構管理、金融市場、投資銀行策略與實務、財金==新聞選讀==、金融==專題研討==
#### ==金融商品及投資==領域
ESG/永續金融: 責任投資、綠色融資與綠色保險、金融領域-證券市場、金融領域-證券市場實務、現代金融實務：資產管理的實戰與挑戰、固定收益證券
#### ==數量方法==領域
金融數量、計量經濟學、==機器學習與財務計量==
#### ==程式設計==領域
金融計算程式運用(一)、金融計算程式運用(二)、計算機程式設計
#### 其它領域
財務==賽局==、金融法規
